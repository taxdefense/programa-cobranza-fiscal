# Landing UI kit — Programa de Cobranza Fiscal

Two directions, both on the design system's real tokens (teal `#0F758A` + Nunito Sans):

- **`index.html`** (current) — premium, institutional redesign: airy whitespace, floating photo cards (real-photo placeholders via `<image-slot>`), one idea per screen, calm color use, no dark/gradient sections, subtle scroll fade-ups. "Entering the firm's program," not buying an infoproduct.
- **`index-v1-curso.html`** (superseded) — the original denser, more sales-page-style recreation with teal/deep-teal bands, stats, chips, sticky mobile CTA bar. Kept for reference/comparison.

Composed entirely from the design system's primitives (`Header`, `Footer`, `Eyebrow`, `Card`, `AccordionItem`, `PriceCard`, `FormField`, `CheckRow`, `Button`, plus `Chip`/`StatBlock`/`Notice`/`Badge` in the v1 file). Click-through: FAQ/module accordions open, inscription form (front-end only, no submission), draggable photo placeholders.
